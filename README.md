# Student Management System(JDBC+MYSQL)
->This Project Student Management System develop using JDBC and MYSQL.
->This Project follws a layered architecture (Model -> DAO -> Service -> Menu -> Main) and performs complete CRUD Operations;

## Features
->Add Student
->View Student
->Search Student by id;
->Search Student by name;
->Search Student by email;
->Update Student;
->Delete Student;
->Email && Phone validation;
->JDBC + MYSQL;

## TECHNOLOGIES USED
->Java8+
->JDBC
->MYSQL
->Intellij Idea
->Git && GitHub;

## PROJECT STRUCTURE
src
|->com.manish
     |->model
     |->dao
     |->service
     |->menu

1.Cone the repository.

5.Run Main.java
